# 图标占位符说明

## 需要准备的图标文件

请在项目根目录下放置以下三个尺寸的图标文件：

| 文件名 | 尺寸 | 用途 |
|--------|------|------|
| icon16.png | 16x16 像素 | Chrome 工具栏小图标 |
| icon48.png | 48x48 像素 | 扩展管理页面图标 |
| icon128.png | 128x128 像素 | Chrome Web Store 图标 |

## 图标设计建议

- **主题**：简洁、专业的屏幕相关图标
- **颜色**：黑色背景 + 白色元素（与插件风格一致）
- **格式**：PNG 格式，支持透明背景
- **设计元素**：可以包含显示器、像素点、屏幕测试相关符号

## 快速替代方案

如果暂时没有图标，可以使用以下方法：

1. 使用在线图标生成工具（如 Canva、Figma）创建简单图标
2. 使用纯色占位图标（黑色方块即可）
3. 从开源图标库（如 Flaticon、Iconfont）下载免费图标

## 临时解决方案

在准备好正式图标之前，可以暂时注释掉 manifest.json 中的 icons 配置，插件仍可正常运行（只是没有图标）。
